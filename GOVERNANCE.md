# Governance

Orbit AI is currently maintained by the repository owner.

## Decision Making

Maintainers prioritize:

1. User safety and privacy.
2. Local-first, auditable behavior.
3. Stable CLI workflows.
4. Tests and reproducible behavior.
5. Small, reviewable changes.

For now, significant changes should start as GitHub issues. Pull requests are reviewed by maintainers before merge.

## Maintainer Responsibilities

- Keep `main` passing `make check`.
- Review security-sensitive changes carefully.
- Keep memory and autonomy behavior explainable.
- Document user-facing configuration and migration behavior.

## Becoming a Maintainer

Regular contributors may be invited to maintain areas they have consistently improved, especially tests, documentation, memory behavior, voice I/O, or backend integrations.
