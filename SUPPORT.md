# Support

Orbit AI is an early-stage open-source project. Support is best-effort.

## Where to Ask

- Bugs: open a GitHub issue with reproduction steps.
- Feature ideas: open a GitHub issue describing the workflow and expected behavior.
- Security issues: follow `SECURITY.md`; do not open public issues.

## Useful Debug Information

When reporting a problem, include:

- Operating system and Python version.
- Whether you use Codex app-server or Ollama.
- Relevant config snippets with secrets removed.
- The command you ran.
- Output from `make check` when applicable.

Do not include API keys, access tokens, private emails, private calendar data, or full personal transcripts.
