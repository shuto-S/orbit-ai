# Roadmap

Orbit AI is moving from a personal MVP toward a reusable local-first assistant framework.

## Near Term

- Improve installation guides for macOS, Ollama, Codex app-server, and VOICEVOX.
- Add more scenario replay tests for memory, autonomy, and proactive behavior.
- Expand privacy controls for memory retention and deletion.
- Add richer examples for Japanese voice workflows.

## Medium Term

- Add optional semantic retrieval while keeping a dependency-free default.
- Support richer local action plugins with explicit permission policies.
- Add evaluation fixtures for memory extraction quality.
- Improve latency instrumentation and voice turn responsiveness.

## Long Term

- Provide reusable components for local-first personal agents.
- Support multiple assistant profiles.
- Add a GUI or web dashboard for memory review and task management.
- Publish stable releases with migration notes.
