# Changelog

All notable changes to Orbit AI are documented here.

## Unreleased

- Added OSS project files: license, contribution guide, security policy, support guide, governance, roadmap, GitHub issue templates, PR template, and CI workflow.
- Added documentation for OpenAI OSS support applications, privacy, setup, and demos.

## 0.1.0 - 2026-05-31

- Added terminal-based assistant loop with startup greeting and explicit end confirmation.
- Added text-first input with optional one-turn voice recognition.
- Added VOICEVOX speech output support.
- Added local SQLite memory, session summaries, tasks, daily review, and proactive checks.
- Added permission policy and typed internal task actions.
- Added Codex app-server backend and Ollama backend.
- Added structured long-term memory extraction with local retrieval and user-controlled remember/forget commands.
